# Theory of GRA-Obnulenka

## GRA Algebra
The General Resonance Architecture (GRA) models reality as a hierarchy of states. The fundamental operation is the GRA product (x), which combines two states into a new state with compressed "ontological weight" (GRA (x) GRA = sqrt(GRA)). This is implemented via non-linear transforms (e.g., `tanh`) that reduce dimensionality and enforce self-similarity.

## Potential and Resonance
States evolve in an ontological potential landscape U(Psi). The dynamics are driven by a resonance operator R, which updates the state along the gradient of the potential:

    dPsi/dt = R(Psi, nabla U) ~ Psi - eta * nabla U

This allows the system to seek stable minima (negentropic states) in the multiverse.

## Foam and Obnulenka
"Foam" represents true contradictions or inconsistencies within a state or simulation. When the foam exceeds a critical threshold, the system undergoes an **Obnulenka** (nullification/reset). The Obnulenka operator O collapses the state to a minimal coherent core, shedding accumulated contradictions and allowing the system to restart its evolution from a stable baseline.
